كورس احتراف CSS

تعلم كيف تبني واجهات احترافية باستخدام CSS.

ما هي لغة CSS؟

اختصار لـ Cascading Style Sheets (أوراق الأنماط المتتالية). هي اللغة المسؤولة عن "شكل" الموقع؛ فإذا كان الـ HTML هو الهيكل العظمي للموقع، فإن CSS هي المظهر، الألوان، الخطوط، وتنسيق العناصر.

لماذا نتعلم CSS؟

فصل المحتوى عن التصميم: يمكنك تغيير شكل موقعك بالكامل فقط بتغيير ملف الـ CSS دون لمس كود الـ HTML. التحكم في الأجهزة المختلفة: بفضل تقنية الـ Media Queries، تجعل موقعك يعمل ببراية على الموبايل، التابلت، والكمبيوتر (Responsive Design). تحسين تجربة المستخدم (UX): من خلال إضافة الحركات (Animations) والتأثيرات التي تجعل التفاعل مع الموقع ممتعاً. CSS تتيح لنا إنشاء تصاميم جذابة ومتجاوبة للمواقع الإلكترونية، مما يحسن تجربة المستخدم ويزيد من فعالية الموقع.

التنسيق باستخدام CSS

التنسيق باستخدام CSS هو الفن الذي يحول هيكل الموقع الجامد إلى واجهة حية وجذابة. لفهم كيف يتم التنسيق، يجب أن ننظر إلى CSS كأداة تتحكم في كل بكسل يراه المستخدم. إليك أهم الركائز التي يقوم عليها التنسيق في CSS: 1. نموذج الصندوق (The Box Model) هذا هو المبدأ الأهم في CSS؛ فالمتصفح يرى كل عنصر (سواء كان نصاً، صورة، أو زر) كأنه صندوق مستطيل. يتكون هذا الصندوق من أربع طبقات: المحتوى (Content): النص أو الصورة نفسها. الحشوة (Padding): المساحة الفارغة "داخل" الصندوق حول المحتوى. الحدود (Border): الإطار المحيط بالحشوة والمحتوى. الهامش (Margin): المساحة الفارغة "خارج" الصندوق التي تفصله عن بقية العناصر. 2. أنظمة تخطيط الصفحة (Layout Systems) قديماً كان التنسيق صعباً، لكن الآن لدينا أدوات قوية جداً لترتيب العناصر: Flexbox: يستخدم لتنسيق العناصر في اتجاه واحد (إما صف أو عمود). هو رائع لتوسط العناصر أو توزيعها بالتساوي داخل القائمة (مثل الـ nav-links في موقعك). CSS Grid: النظام الأقوى لتصميم صفحات كاملة ثنائية الأبعاد (صفوف وأعمدة معاً). هو الخيار الأمثل لعمل "شبكة الكورسات" التي وضعتها في مشروعك. 3. الألوان والتدرجات (Colors & Gradients) CSS تمنحك تحكماً كاملاً في الألوان باستخدام: الأسماء: مثل gold أو black. نظام HEX: مثل #ffd117 (الذي استخدمته أنت للذهبي). التدرجات (Gradients): يمكنك دمج لونين أو أكثر ليظهرا بشكل انسيابي خلف الكروت أو الأزرار لإعطاء لمسة عصرية. 4. التنسيق المتجاوب (Responsive Design) بما أن الناس يتصفحون من الموبايل والكمبيوتر، نستخدم ما يسمى Media Queries. وهي عبارة عن "شروط" نخبر بها المتصفح: "إذا كانت شاشة المستخدم أصغر من 600 بكسل، اجعل حجم الخط أصغر واجعل الكورسات تظهر فوق بعضها بدلاً من بجانب بعضها". 5. الحركات والتفاعلات (Transitions & Animations) التنسيق لا يتوقف عند الثبات؛ يمكنك جعل الموقع "يتنفس" عن طريق: Hover Effects: تغيير لون الزر أو تكبير الكارت قليلاً عندما يمرر المستخدم الماوس فوقه.
